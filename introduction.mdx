## Welcome to Viggle API

Generate AI-powered character animation videos. Submit pre-processed character and scene IDs, then render videos with two modes: progressive chunk delivery for real-time playback, or batch processing for complete video download.

<CardGroup cols={2}>
  <Card title="Live Render" icon="bolt">
    Stream video chunks as they're generated. Ideal for real-time preview and progressive playback.
  </Card>
  <Card title="Batch Render" icon="layer-group">
    Wait for complete video. Simple workflow for background processing and file downloads.
  </Card>
</CardGroup>

## Base URL

All API requests should be made to:

```
https://api.viggle.ai
```

## Quick Start

Generate your first AI animation video in 3 steps:

1. **Get API Key** - Create one in your [Dashboard](https://viggle.ai/dashboard)
2. **Render** - Submit a character ID + scene ID via `POST /api/render`
3. **Download** - Get your video from `/api/render/{job_id}/download`

<Note>
  Characters and scenes are preprocessed via the Viggle platform. You'll use their IDs (`char_xxx`, `scene_xxx`) when submitting render jobs through the API.
</Note>

### Complete Example

Copy and run this script. Replace `YOUR_API_KEY`, `char_xxx`, and `scene_xxx` with your actual values.

<CodeGroup>

```python Python
import requests
import time

API_KEY = "YOUR_API_KEY"
BASE = "https://api.viggle.ai"
headers = {"Authorization": f"Bearer {API_KEY}"}

# Step 1: Start Render
print("1. Starting render...")
resp = requests.post(
    f"{BASE}/api/render",
    headers=headers,
    data={
        "character_id": "char_xxx",
        "scene_id": "scene_xxx"
    }
)
job_id = resp.json()["job_id"]
print(f"   Job ID: {job_id}")

# Step 2: Poll status until complete (no auth needed for status)
print("2. Waiting for render to complete...")
while True:
    status = requests.get(f"{BASE}/api/render/{job_id}").json()
    progress = status.get("progress", {})
    completed = progress.get("completed_chunks", 0)
    total = progress.get("total_chunks", 0)
    print(f"   Status: {status['status']} ({completed}/{total})")

    if status["status"] == "complete":
        print(f"   Video ready: {status['cdn_url']}")
        break
    elif status["status"] == "failed":
        raise Exception(status.get("error", "Render failed"))

    time.sleep(5)

# Step 3: Download
print("3. Downloading video...")
video = requests.get(
    f"{BASE}/api/render/{job_id}/download",
    headers=headers,
    allow_redirects=True
)
with open("output.mp4", "wb") as f:
    f.write(video.content)
print("Done! Saved to output.mp4")
```


```javascript JavaScript
const fs = require('fs');

const API_KEY = "YOUR_API_KEY";
const BASE = "https://api.viggle.ai";
const headers = { "Authorization": `Bearer ${API_KEY}` };

const sleep = (ms) => new Promise(r => setTimeout(r, ms));

async function main() {
  // Step 1: Start Render
  console.log("1. Starting render...");
  const form = new URLSearchParams();
  form.append("character_id", "char_xxx");
  form.append("scene_id", "scene_xxx");

  let resp = await fetch(`${BASE}/api/render`, {
    method: "POST",
    headers,
    body: form
  }).then(r => r.json());
  const jobId = resp.job_id;
  console.log(`   Job ID: ${jobId}`);

  // Step 2: Poll status until complete (no auth needed for status)
  console.log("2. Waiting for render to complete...");
  while (true) {
    const status = await fetch(`${BASE}/api/render/${jobId}`).then(r => r.json());
    const completed = status.progress?.completed_chunks ?? 0;
    const total = status.progress?.total_chunks ?? 0;
    console.log(`   Status: ${status.status} (${completed}/${total})`);

    if (status.status === "complete") {
      console.log(`   Video ready: ${status.cdn_url}`);
      break;
    }
    if (status.status === "failed") throw new Error(status.error || "Render failed");
    await sleep(5000);
  }

  // Step 3: Download
  console.log("3. Downloading video...");
  const video = await fetch(`${BASE}/api/render/${jobId}/download`, { headers });
  fs.writeFileSync("output.mp4", Buffer.from(await video.arrayBuffer()));
  console.log("Done! Saved to output.mp4");
}

main().catch(console.error);
```

## Rendering Modes

### Live Render (Progressive)

Use the **Poll endpoint** with long-polling to receive video chunks as they're rendered. Best for:

- Real-time preview
- Progressive playback
- Streaming applications

### Batch Render (Complete)

Use the **Status endpoint** (no auth required) to check progress and download the complete video when ready. Best for:

- Background processing
- File downloads
- Simple integration
