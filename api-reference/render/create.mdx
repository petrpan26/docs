title: 'Create Render Job'
api: 'POST /api/render'
description: 'Start a video render job'
---

Start a video render job. Both character and scene must have status `ready`.

## Request

<ParamField body="character_id" type="string" required>
  ID of a ready character.
</ParamField>

<ParamField body="scene_id" type="string" required>
  ID of a ready scene.
</ParamField>

<ParamField body="options" type="object">
  Render options. See [Render Options](/render-options) for all available options.
  
  <Expandable title="Options">
    <ParamField body="options.model" type="string" default="V4">
      `V4` (better quality) or `V3` (faster)
    </ParamField>
    <ParamField body="options.background" type="string" default="inpaint">
      `inpaint`, `original`, `solid`, or `transparent`
    </ParamField>
    <ParamField body="options.background_color" type="array">
      RGB array for solid background, e.g., `[0, 255, 0]`
    </ParamField>
    <ParamField body="options.fps" type="integer">
      Output frame rate (`null` matches input)
    </ParamField>
    <ParamField body="options.fast" type="boolean" default="false">
      Enable parallel processing for ~2x speed
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="job_id" type="string">
  Unique identifier for the render job
</ResponseField>

<ResponseField name="status" type="string">
  Initial status will be `queued`
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST "https://api.viggle.ai/api/render" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "character_id": "char_550e8400e29b41d4",
    "scene_id": "scene_660e8400e29b41d4",
    "options": {
      "model": "V4",
      "background": "inpaint"
    }
  }'
```

```python Python
import requests

response = requests.post(
    "https://api.viggle.ai/api/render",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "character_id": "char_550e8400e29b41d4",
        "scene_id": "scene_660e8400e29b41d4",
        "options": {"model": "V4", "background": "inpaint"}
    }
)
job = response.json()
print(f"Job ID: {job['job_id']}")
```

```javascript JavaScript
const response = await fetch("https://api.viggle.ai/api/render", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    character_id: "char_550e8400e29b41d4",
    scene_id: "scene_660e8400e29b41d4",
    options: { model: "V4", background: "inpaint" }
  })
});
const job = await response.json();
console.log(`Job ID: ${job.job_id}`);
```

## Next Steps

After creating a render job, you can either:

- **Live Render**: Use [Poll for Chunks](/api-reference/render/poll) to receive video chunks progressively
- **Batch Render**: Use [Get Job Status](/api-reference/render/status) to check progress, then [Download Video](/api-reference/render/download) when complete
