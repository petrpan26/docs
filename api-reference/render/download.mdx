---
title: 'Download Video'
api: 'GET /api/render/{job_id}/download'
description: 'Download the completed video'
---

Download the completed video. Only available when job status is `complete`. Returns a `302` redirect to the CDN URL where the video is hosted.

<Note>
  This endpoint requires authentication. Include your API key in the `Authorization: Bearer` header.
</Note>

## Path Parameters

<ParamField path="job_id" type="string" required>
  The render job ID
</ParamField>

## Response

Returns a `302 Found` redirect to the CDN URL. Most HTTP clients will follow the redirect automatically and return the binary MP4 data.

<RequestExample>

```bash cURL
# Follow redirect automatically with -L
curl -L "https://api.viggle.ai/api/render/job_abc123xyz/download" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -o output.mp4
```

```python Python
import requests

API_KEY = "your-api-key"
job_id = "job_abc123xyz"

# requests follows redirects by default
video = requests.get(
    f"https://api.viggle.ai/api/render/{job_id}/download",
    headers={"Authorization": f"Bearer {API_KEY}"}
)

# Save to file
with open("output.mp4", "wb") as f:
    f.write(video.content)

print("Video saved to output.mp4")
```

```javascript JavaScript
const jobId = "job_abc123xyz";

const response = await fetch(
  `https://api.viggle.ai/api/render/${jobId}/download`,
  { headers: { "Authorization": `Bearer ${API_KEY}` } }
);

const blob = await response.blob();
const url = URL.createObjectURL(blob);

// Create download link
const a = document.createElement('a');
a.href = url;
a.download = 'output.mp4';
a.click();
```

</RequestExample>

## Complete Batch Render Example

<CodeGroup>

```python Python
import requests
import time

API_KEY = "your-api-key"
BASE = "https://api.viggle.ai"
headers = {"Authorization": f"Bearer {API_KEY}"}

# 1. Create render job
job = requests.post(f"{BASE}/api/render", headers=headers, data={
    "character_id": "char_xxx",
    "scene_id": "scene_xxx"
}).json()

print(f"Job started: {job['job_id']}")

# 2. Poll until complete (status endpoint is open -- no auth needed)
while True:
    status = requests.get(f"{BASE}/api/render/{job['job_id']}").json()
    progress = status.get("progress", {})
    completed = progress.get("completed_chunks", 0)
    total = progress.get("total_chunks", 0)

    print(f"Status: {status['status']} ({completed}/{total})")

    if status["status"] == "complete":
        break
    elif status["status"] == "failed":
        raise Exception(status["error"])

    time.sleep(5)

# 3. Download video (auth required)
video = requests.get(
    f"{BASE}/api/render/{job['job_id']}/download",
    headers=headers
)
with open("output.mp4", "wb") as f:
    f.write(video.content)

print("Video saved!")
```

```javascript JavaScript
const API_KEY = "your-api-key";
const BASE = "https://api.viggle.ai";

// 1. Create render job
const form = new URLSearchParams();
form.append("character_id", "char_xxx");
form.append("scene_id", "scene_xxx");

const job = await fetch(`${BASE}/api/render`, {
  method: "POST",
  headers: { "Authorization": `Bearer ${API_KEY}` },
  body: form
}).then(r => r.json());

console.log(`Job started: ${job.job_id}`);

// 2. Poll until complete (status endpoint is open -- no auth needed)
while (true) {
  const status = await fetch(`${BASE}/api/render/${job.job_id}`).then(r => r.json());
  const completed = status.progress?.completed_chunks ?? 0;
  const total = status.progress?.total_chunks ?? 0;

  console.log(`Status: ${status.status} (${completed}/${total})`);

  if (status.status === "complete") break;
  if (status.status === "failed") throw new Error(status.error);

  await new Promise(r => setTimeout(r, 5000));
}

// 3. Download video (auth required)
const video = await fetch(
  `${BASE}/api/render/${job.job_id}/download`,
  { headers: { "Authorization": `Bearer ${API_KEY}` } }
).then(r => r.blob());

// Save or display the video
const url = URL.createObjectURL(video);
console.log(`Video URL: ${url}`);
```
