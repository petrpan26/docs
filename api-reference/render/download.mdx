---
title: 'Download Video'
icon: 'download'
openapi: 'GET /api/render/{job_id}/download'
description: 'Download the completed video'
---

Download the completed video. Only available when job status is `complete`. Returns a `302` redirect to the CDN URL where the video is hosted. The downloaded file is a standard MP4 with `faststart` (moov atom at the beginning), not the [fMP4 format](/guides/fmp4-format) used for individual chunks.

<Info>
  **Transparent Mode**: When `background_mode=transparent` was used, the poll response includes a `mask_cdn_url` field with a direct link to the mask video. The mask is a grayscale video (white = character, black = background) for compositing in video editors.
</Info>

## Examples

<CodeGroup>
```python Python
import requests

job_id = "job_abc123xyz"
headers = {"Authorization": "Bearer YOUR_API_KEY"}

resp = requests.get(
    f"https://apis.viggle.ai/api/render/{job_id}/download",
    headers=headers,
    allow_redirects=True,
    stream=True,
)

if resp.status_code == 200:
    with open("output.mp4", "wb") as f:
        for chunk in resp.iter_content(chunk_size=8192):
            f.write(chunk)
    print("Downloaded output.mp4")
else:
    data = resp.json()
    if "detail" in data:
        print(f"Error {data['detail']['error_code']}: {data['detail']['message']}")
    else:
        print(f"Error: {data}")
```

```javascript JavaScript
const jobId = "job_abc123xyz";

const resp = await fetch(`https://apis.viggle.ai/api/render/${jobId}/download`, {
  headers: { "Authorization": "Bearer YOUR_API_KEY" },
});

if (resp.ok) {
  const blob = await resp.blob();
  // Node.js: write to file
  const fs = await import("fs");
  fs.writeFileSync("output.mp4", Buffer.from(await blob.arrayBuffer()));
  console.log("Downloaded output.mp4");
} else {
  const data = await resp.json();
  if (data.detail) {
    console.error(`Error ${data.detail.error_code}: ${data.detail.message}`);
  }
}
```
