---
title: 'Get Job Status'
icon: 'activity'
openapi: 'GET /api/render/{job_id}'
description: 'Check render job status and progress'
---

Check job status. Returns immediately. Poll every 5 seconds until `complete` or `failed`.

<Note>
  This endpoint does **not** require authentication. You can poll job status without an API key.
</Note>
