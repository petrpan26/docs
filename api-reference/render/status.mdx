---
title: 'Get Job Status'
icon: 'activity'
openapi: 'GET /api/render/{job_id}'
description: 'Check render job status and progress'
---

Check job status. Returns immediately. Poll every 5 seconds until `complete` or `failed`.

When a job fails, the response includes `error` and `error_code` fields:

```json
{
  "status": "failed",
  "error": "Processing resources exceeded",
  "error_code": "CS-RNRD-003",
  "progress": { "completed_chunks": 15, "total_chunks": 22 }
}
```

Include the `error_code` when contacting **support@viggle.ai**. See [Error Codes](/error-codes) for the full list.

## Examples

<CodeGroup>
```python Python
import requests
import time

job_id = "job_abc123xyz"
headers = {"Authorization": "Bearer YOUR_API_KEY"}
url = f"https://apis.viggle.ai/api/render/{job_id}"

while True:
    data = requests.get(url, headers=headers).json()
    status = data["status"]
    print(f"Status: {status}")

    if status == "complete":
        print(f"Video ready: {data.get('cdn_url')}")
        break
    elif status == "failed":
        print(f"Error {data.get('error_code')}: {data.get('error')}")
        break

    time.sleep(5)
```

```javascript JavaScript
const jobId = "job_abc123xyz";
const headers = { "Authorization": "Bearer YOUR_API_KEY" };
const url = `https://apis.viggle.ai/api/render/${jobId}`;

while (true) {
  const resp = await fetch(url, { headers });
  const data = await resp.json();
  console.log(`Status: ${data.status}`);

  if (data.status === "complete") {
    console.log(`Video ready: ${data.cdn_url}`);
    break;
  } else if (data.status === "failed") {
    console.error(`Error ${data.error_code}: ${data.error}`);
    break;
  }

  await new Promise(r => setTimeout(r, 5000));
}
```
