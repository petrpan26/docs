---
title: 'Get Job Status (Batch Render)'
api: 'GET /api/render/{job_id}'
description: 'Check job status for batch rendering'
---

Check job status. Returns immediately. Poll every 5 seconds until `complete` or `failed`.

<Note>
  This endpoint does **not** require authentication. You can poll job status without an API key.
</Note>

## Path Parameters

<ParamField path="job_id" type="string" required>
  The render job ID
</ParamField>

## Response

<ResponseField name="job_id" type="string">
  The job identifier
</ResponseField>

<ResponseField name="status" type="string">
  Current status: `queued`, `processing`, `rendering`, `complete`, or `failed`
</ResponseField>

<ResponseField name="progress" type="object">
  Rendering progress

  <Expandable title="Progress object">
    <ResponseField name="completed_chunks" type="integer">
      Number of completed chunks
    </ResponseField>
    <ResponseField name="total_chunks" type="integer">
      Total number of chunks
    </ResponseField>
    <ResponseField name="percent" type="number">
      Completion percentage (0-100)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="cdn_url" type="string">
  CDN URL to download complete video (available when status is `complete`)
</ResponseField>

<ResponseField name="chunks" type="array">
  Array of individual chunks with their CDN URLs

  <Expandable title="Chunk object">
    <ResponseField name="chunk_id" type="integer">
      Chunk identifier
    </ResponseField>
    <ResponseField name="cdn_url" type="string">
      CDN URL for this chunk
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string">
  Error message (when status is `failed`)
</ResponseField>

<RequestExample>

```bash cURL
curl "https://api.viggle.ai/api/render/job_abc123xyz"
```

```python Python
import requests
import time

BASE = "https://api.viggle.ai"

job_id = "job_abc123xyz"

while True:
    status = requests.get(f"{BASE}/api/render/{job_id}").json()
    progress = status.get("progress", {})
    completed = progress.get("completed_chunks", 0)
    total = progress.get("total_chunks", 0)
    percent = progress.get("percent", 0)

    print(f"Status: {status['status']} ({completed}/{total}, {percent:.0f}%)")

    if status["status"] == "complete":
        print(f"Video ready: {status['cdn_url']}")
        for chunk in status.get("chunks", []):
            print(f"  Chunk {chunk['chunk_id']}: {chunk['cdn_url']}")
        break
    elif status["status"] == "failed":
        raise Exception(status["error"])

    time.sleep(5)
```

```javascript JavaScript
const jobId = "job_abc123xyz";

async function waitForComplete() {
  while (true) {
    const response = await fetch(
      `https://api.viggle.ai/api/render/${jobId}`
    );
    const status = await response.json();
    const completed = status.progress?.completed_chunks ?? 0;
    const total = status.progress?.total_chunks ?? 0;
    const percent = status.progress?.percent ?? 0;

    console.log(`Status: ${status.status} (${completed}/${total}, ${percent.toFixed(0)}%)`);

    if (status.status === "complete") {
      console.log(`Video ready: ${status.cdn_url}`);
      for (const chunk of status.chunks || []) {
        console.log(`  Chunk ${chunk.chunk_id}: ${chunk.cdn_url}`);
      }
      return status;
    } else if (status.status === "failed") {
      throw new Error(status.error);
    }

    await new Promise(r => setTimeout(r, 5000));
  }
}
```
