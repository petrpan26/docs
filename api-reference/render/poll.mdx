---
title: 'Poll for Chunks (Live Render)'
openapi: 'GET /api/render/{job_id}/poll'
description: 'Long-poll for video chunks as they render'
---

Long-poll for new video chunks. Blocks until chunks are ready, job completes, or timeout. Use for real-time progressive playback.

Each chunk is a [fragmented MP4 (fMP4)](/guides/fmp4-format) file that can be played immediately via [Media Source Extensions (MSE)](/guides/live-render#progressive-playback-with-fmp4--mse) or saved to disk. See the [Live Rendering guide](/guides/live-render) for complete streaming examples.
