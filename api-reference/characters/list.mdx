Retrieve a list of all characters associated with your account, ordered by most recent first.

## Example

<CodeGroup>
```python Python
import requests

url = "https://apis.viggle.ai/api/characters"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.get(url, headers=headers)
response.raise_for_status()
result = response.json()

for character in result:
    print(f"{character['id']}: {character['name']} ({character['status']})")
```

```javascript JavaScript
const response = await fetch("https://apis.viggle.ai/api/characters", {
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (!response.ok) throw new Error(`HTTP ${response.status}`);
result.forEach((character) => {
  console.log(`${character.id}: ${character.name} (${character.status})`);
});
```

```bash cURL
curl -X GET https://apis.viggle.ai/api/characters \
  -H "Authorization: Bearer YOUR_API_KEY"
```
