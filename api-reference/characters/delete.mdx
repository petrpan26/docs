Permanently delete a character. This action cannot be undone.

<Warning>
  Deleting a character will not affect previously rendered videos, but you won't be able to use this character in new render jobs.
</Warning>

## Example

<CodeGroup>
```python Python
import requests

character_id = "YOUR_CHARACTER_ID"
url = f"https://apis.viggle.ai/api/characters/{character_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.delete(url, headers=headers)
response.raise_for_status()
result = response.json()

    print("Character deleted successfully")
```

```javascript JavaScript
const characterId = "YOUR_CHARACTER_ID";

const response = await fetch(`https://apis.viggle.ai/api/characters/${characterId}`, {
  method: "DELETE",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (!response.ok) throw new Error(`HTTP ${response.status}`);
console.log("Character deleted successfully");
```

```bash cURL
curl -X DELETE https://apis.viggle.ai/api/characters/YOUR_CHARACTER_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
```
