---
title: 'Get Character'
api: 'GET /api/characters/{id}'
description: 'Get character details by ID'
---

Retrieve details for a specific character. Use this endpoint to poll for processing status.

<Note>
  Poll every 5 seconds until status is `ready` or `failed`.
</Note>

## Path Parameters

<ParamField path="id" type="string" required>
  The character ID (e.g., `char_550e8400e29b41d4`)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique character identifier
</ResponseField>

<ResponseField name="name" type="string">
  Display name
</ResponseField>

<ResponseField name="status" type="string">
  Processing status:
  - `pending` - Queued for processing
  - `processing` - Currently being processed
  - `ready` - Available for use in renders
  - `failed` - Processing failed (check `error` field)
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  URL to character thumbnail image (available when `ready`)
</ResponseField>

<ResponseField name="error" type="string">
  Error message if status is `failed`, otherwise `null`
</ResponseField>

<ResponseField name="credits_used" type="integer">
  Credits consumed
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 creation timestamp
</ResponseField>

<RequestExample>

```bash cURL
curl "https://apis.viggle.ai/api/characters/char_550e8400e29b41d4" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests
import time

character_id = "char_550e8400e29b41d4"

# Poll until ready
while True:
    response = requests.get(
        f"https://apis.viggle.ai/api/characters/{character_id}",
        headers={"Authorization": "Bearer YOUR_API_KEY"}
    )
    data = response.json()
    
    if data["status"] == "ready":
        print("Character ready!")
        break
    elif data["status"] == "failed":
        raise Exception(data["error"])
    
    time.sleep(5)
```

```javascript JavaScript
const characterId = "char_550e8400e29b41d4";

async function waitForCharacter() {
  while (true) {
    const response = await fetch(
      `https://apis.viggle.ai/api/characters/${characterId}`,
      { headers: { "Authorization": `Bearer ${API_KEY}` } }
    );
    const data = await response.json();
    
    if (data.status === "ready") return data;
    if (data.status === "failed") throw new Error(data.error);
    
    await new Promise(r => setTimeout(r, 5000));
  }
}
```
