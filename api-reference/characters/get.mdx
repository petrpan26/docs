Retrieve details for a specific character. Use this endpoint to poll for processing status.

<Note>
  Poll every 5 seconds until status is `ready` or `failed`.
</Note>

## Example

<CodeGroup>
```python Python
import requests

character_id = "YOUR_CHARACTER_ID"
url = f"https://apis.viggle.ai/api/characters/{character_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.get(url, headers=headers)
result = response.json()

if result.get("error_code"):
    print(f"Error: {result['error_code']} - {result.get('message')}")
else:
    character = result["data"]
    print(f"Name: {character['name']}")
    print(f"Status: {character['status']}")
```

```javascript JavaScript
const characterId = "YOUR_CHARACTER_ID";

const response = await fetch(`https://apis.viggle.ai/api/characters/${characterId}`, {
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (result.error_code) {
  console.error(`Error: ${result.error_code} - ${result.message}`);
} else {
  console.log(`Name: ${result.data.name}`);
  console.log(`Status: ${result.data.status}`);
}
```

```bash cURL
curl -X GET https://apis.viggle.ai/api/characters/YOUR_CHARACTER_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
```
