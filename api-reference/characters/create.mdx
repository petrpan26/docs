---

Upload a reference image to create a reusable character. Processing runs in background. **Costs 1 credit.**

<Note>
  Best results with clear, front-facing subjects with good lighting and high resolution.
</Note>

## Next Steps

After creating a character, poll [Get Character](/api-reference/characters/get) every 5 seconds until the status becomes `ready`. Then you can use it in [render jobs](/api-reference/render/create).
