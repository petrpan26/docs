title: 'Create Scene'
api: 'POST /api/scenes/preprocess'
description: 'Upload a driving video to create a reusable scene'
---

Upload a driving video to create a reusable scene. The scene extracts motion data that can be applied to characters. **Costs 1 credit per second of video.**

## Request

<ParamField body="name" type="string" required>
  Display name for the scene.
</ParamField>

<ParamField body="video" type="file" required>
  Driving video file. Supported formats:
  - MP4
  - MOV
  - WebM
  
  The video contains the motion that will be transferred to characters.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the scene (e.g., `scene_660e8400e29b41d4`)
</ResponseField>

<ResponseField name="status" type="string">
  Initial status will be `pending`. Poll [Get Scene](/api-reference/scenes/get) until `ready` or `failed`.
</ResponseField>

<ResponseField name="credits_reserved" type="integer">
  Number of credits reserved (based on video duration)
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST "https://apis.viggle.ai/api/scenes/preprocess" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "name=Dance Sequence" \
  -F "video=@dance.mp4"
```

```python Python
import requests

response = requests.post(
    "https://apis.viggle.ai/api/scenes/preprocess",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    files={"video": open("dance.mp4", "rb")},
    data={"name": "Dance Sequence"}
)
print(response.json())
```

## Next Steps

After creating a scene, poll [Get Scene](/api-reference/scenes/get) every 5 seconds until the status becomes `ready`. Then you can use it in [render jobs](/api-reference/render/create).
