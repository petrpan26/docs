Retrieve a list of all scenes associated with your account, ordered by most recent first.

## Example

<CodeGroup>
```python Python
import requests

url = "https://apis.viggle.ai/api/scenes"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.get(url, headers=headers)
result = response.json()

if result.get("error_code"):
    print(f"Error: {result['error_code']} - {result.get('message')}")
else:
    for scene in result["data"]:
        print(f"{scene['id']}: {scene['name']} ({scene['status']})")
```

```javascript JavaScript
const response = await fetch("https://apis.viggle.ai/api/scenes", {
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (result.error_code) {
  console.error(`Error: ${result.error_code} - ${result.message}`);
} else {
  result.data.forEach((scene) => {
    console.log(`${scene.id}: ${scene.name} (${scene.status})`);
  });
}
```

```bash cURL
curl -X GET https://apis.viggle.ai/api/scenes \
  -H "Authorization: Bearer YOUR_API_KEY"
```
