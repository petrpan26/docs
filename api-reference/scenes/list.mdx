Retrieve a list of all scenes associated with your account, ordered by most recent first.

## Example

<CodeGroup>
```python Python
import requests

url = "https://apis.viggle.ai/api/scenes"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.get(url, headers=headers)
response.raise_for_status()
result = response.json()

for scene in result:
    print(f"{scene['id']}: {scene['name']} ({scene['status']})")
```

```javascript JavaScript
const response = await fetch("https://apis.viggle.ai/api/scenes", {
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (!response.ok) throw new Error(`HTTP ${response.status}`);
result.forEach((scene) => {
  console.log(`${scene.id}: ${scene.name} (${scene.status})`);
});
```

```bash cURL
curl -X GET https://apis.viggle.ai/api/scenes \
  -H "Authorization: Bearer YOUR_API_KEY"
```
