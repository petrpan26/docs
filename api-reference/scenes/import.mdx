---
title: 'Import Multi-Person Template'
api: 'POST /api/scenes/import'
description: 'Import a multi-person scene from a Viggle template'
---

Import a multi-person scene from a Viggle template. Multi-person templates contain pre-tracked segments for each person in the video, enabling multi-character rendering.

<Info>
  **Multi-Character Scenes**
  
  Multi-person templates contain person segments with unique UUIDs. Use these UUIDs in [Multi-Character Render](/api-reference/render/multi-character) to map different avatars to each person.
</Info>

## Request

<ParamField body="template_id" type="string" required>
  Viggle multi-person template UUID from your library.
</ParamField>

<ParamField body="name" type="string">
  Optional display name for the scene.
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the imported scene
</ResponseField>

<ResponseField name="status" type="string">
  Initial status will be `pending`
</ResponseField>

<ResponseField name="credits_reserved" type="integer">
  Number of credits reserved
</ResponseField>

<ResponseField name="segments" type="array">
  Array of tracked person segments. Each segment has:
  
  <Expandable title="Segment object">
    <ResponseField name="uuid" type="string">
      Unique identifier for this person (e.g., `person_a1b2c3d4`)
    </ResponseField>
    <ResponseField name="type" type="string">
      Segment type (typically `human`)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST "https://apis.viggle.ai/api/scenes/import" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "template_id": "template_uuid",
    "name": "Group Dance"
  }'
```

```python Python
import requests

response = requests.post(
    "https://apis.viggle.ai/api/scenes/import",
    headers={
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "template_id": "template_uuid",
        "name": "Group Dance"
    }
)
data = response.json()

# Get segment UUIDs for character mapping
for segment in data["segments"]:
    print(f"Person: {segment['uuid']}")
```

```javascript JavaScript
const response = await fetch("https://apis.viggle.ai/api/scenes/import", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${API_KEY}`,
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    template_id: "template_uuid",
    name: "Group Dance"
  })
});
const data = await response.json();

// Get segment UUIDs for character mapping
data.segments.forEach(segment => {
  console.log(`Person: ${segment.uuid}`);
});
```

</RequestExample>

## Using Segment UUIDs

The `segments` array contains UUIDs for each tracked person. Use these UUIDs in the `character_mapping` parameter when [creating a multi-character render](/api-reference/render/multi-character).

```json
{
  "scene_id": "scene_770e8400e29b41d4",
  "character_mapping": {
    "person_a1b2c3d4": "char_alice",
    "person_e5f6g7h8": "char_bob"
  }
}
```
