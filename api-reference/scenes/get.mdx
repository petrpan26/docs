---
title: 'Get Scene'
api: 'GET /api/scenes/{id}'
description: 'Get scene details by ID'
---

Retrieve details for a specific scene. Use this endpoint to poll for processing status.

<Note>
  Poll every 5 seconds until status is `ready` or `failed`.
</Note>

## Path Parameters

<ParamField path="id" type="string" required>
  The scene ID (e.g., `scene_660e8400e29b41d4`)
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique scene identifier
</ResponseField>

<ResponseField name="name" type="string">
  Display name
</ResponseField>

<ResponseField name="status" type="string">
  Processing status:
  - `pending` - Queued for processing
  - `processing` - Currently being processed
  - `ready` - Available for use in renders
  - `failed` - Processing failed (check `error` field)
</ResponseField>

<ResponseField name="duration_seconds" type="number">
  Video duration in seconds
</ResponseField>

<ResponseField name="fps" type="integer">
  Frames per second
</ResponseField>

<ResponseField name="thumbnail_url" type="string">
  URL to scene thumbnail (available when `ready`)
</ResponseField>

<ResponseField name="segments" type="array">
  Array of person segments for multi-person template scenes. `null` for regular scenes.
  
  <Expandable title="Segment object">
    <ResponseField name="uuid" type="string">
      Unique identifier for this person
    </ResponseField>
    <ResponseField name="type" type="string">
      Segment type (typically `human`)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="error" type="string">
  Error message if status is `failed`, otherwise `null`
</ResponseField>

<ResponseField name="credits_used" type="integer">
  Credits consumed
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 creation timestamp
</ResponseField>

<RequestExample>

```bash cURL
curl "https://api.viggle.ai/api/scenes/scene_660e8400e29b41d4" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests
import time

scene_id = "scene_660e8400e29b41d4"

# Poll until ready
while True:
    response = requests.get(
        f"https://api.viggle.ai/api/scenes/{scene_id}",
        headers={"Authorization": "Bearer YOUR_API_KEY"}
    )
    data = response.json()
    
    if data["status"] == "ready":
        print("Scene ready!")
        break
    elif data["status"] == "failed":
        raise Exception(data["error"])
    
    time.sleep(5)
```

```javascript JavaScript
const sceneId = "scene_660e8400e29b41d4";

async function waitForScene() {
  while (true) {
    const response = await fetch(
      `https://api.viggle.ai/api/scenes/${sceneId}`,
      { headers: { "Authorization": `Bearer ${API_KEY}` } }
    );
    const data = await response.json();
    
    if (data.status === "ready") return data;
    if (data.status === "failed") throw new Error(data.error);
    
    await new Promise(r => setTimeout(r, 5000));
  }
}
```
