openapi: 'DELETE /api/scenes/{scene_id}'
description: 'Permanently delete a scene'
---

Permanently delete a scene. This action cannot be undone.

<Warning>
  Deleting a scene will not affect previously rendered videos, but you won't be able to use this scene in new render jobs.
</Warning>

## Example

<CodeGroup>
```python Python
import requests

scene_id = "YOUR_SCENE_ID"
url = f"https://apis.viggle.ai/api/scenes/{scene_id}"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.delete(url, headers=headers)
response.raise_for_status()
result = response.json()

    print("Scene deleted successfully")
```

```javascript JavaScript
const sceneId = "YOUR_SCENE_ID";

const response = await fetch(`https://apis.viggle.ai/api/scenes/${sceneId}`, {
  method: "DELETE",
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (!response.ok) throw new Error(`HTTP ${response.status}`);
console.log("Scene deleted successfully");
```

```bash cURL
curl -X DELETE https://apis.viggle.ai/api/scenes/YOUR_SCENE_ID \
  -H "Authorization: Bearer YOUR_API_KEY"
```
