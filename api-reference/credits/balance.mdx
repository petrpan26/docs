Retrieve the current credit balance for your account.

## Example

<CodeGroup>
```python Python
import requests

url = "https://apis.viggle.ai/api/credits"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
}

response = requests.get(url, headers=headers)
result = response.json()

if result.get("error_code"):
    print(f"Error: {result['error_code']} - {result.get('message')}")
else:
    print(f"Credit balance: {result['data']['balance']}")
```

```javascript JavaScript
const response = await fetch("https://apis.viggle.ai/api/credits", {
  headers: {
    "Authorization": "Bearer YOUR_API_KEY",
  },
});

const result = await response.json();

if (result.error_code) {
  console.error(`Error: ${result.error_code} - ${result.message}`);
} else {
  console.log(`Credit balance: ${result.data.balance}`);
}
```

```bash cURL
curl -X GET https://apis.viggle.ai/api/credits \
  -H "Authorization: Bearer YOUR_API_KEY"
```
