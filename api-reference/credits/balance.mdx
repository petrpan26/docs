---
title: 'Get Credit Balance'
api: 'GET /api/credits'
description: 'Check your remaining credit balance'
---

Retrieve the current credit balance for your account.

## Response

<ResponseField name="credits_available" type="number">
  Number of credits remaining in your account
</ResponseField>

<ResponseField name="user_id" type="string">
  Your unique user identifier
</ResponseField>

<RequestExample>

```bash cURL
curl "https://apis.viggle.ai/api/credits" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://apis.viggle.ai/api/credits",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch("https://apis.viggle.ai/api/credits", {
  headers: {
    "Authorization": `Bearer ${API_KEY}`
  }
});
const data = await response.json();
```

</RequestExample>
